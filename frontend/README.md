# NeverNet Frontend

React + TypeScript + Vite client for the NeverNet event planning system.

## Commands

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Environment

Copy `.env.example` to `.env` and point `VITE_API_BASE_URL` at the backend API.

Default local value:

```bash
VITE_API_BASE_URL=http://localhost:8080/api
```
